VS CODE

Visual Studio Code, commonly referred to as VS Code, is an integrated development environment developed by Microsoft for Windows, Linux, macOS and web browsers. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git

FEATURES OF VS CODE

  • Syntax Highlighting and Auto-Completion for most languages
  • Version Control Integration for Git and other systems.
  • Extensions Marketplace to add functionality, such as language-specific support, linters, debuggers, themes, and more.
  • Integrated Terminal for running scripts or commands.
  • Customizable Interface with user-defined settings and shortcuts.
  • Built-in debugger for your code, allowing you to set breakpoints, inspect variables, and step through your code. Free to use and modify.

    BASIC SHORTCUTS

    >
    ctrl+s save the current file
    ctrl+c copy the selected text or item
    ctrl+o opens the selected file
    ctrl+v paste the copied content
    ctrl+x cut the selected text or item
    ctrl+z undo the last action
    ctrl+y redo the previously undone action
    ctrl+a select all content in the current file
    ctrl+plus(+) zoom in the editor view
    ctrl+minus(-) zoom out the editer view
    ctrl+shift+alt+down add cursors to multiple lines for simulataneous editing
    alt+up+arrow move the selected line or block of code one line down
    alt+down arrow move the selected line or block of code one line up
    ctrl+shift+p open command palette
    ctrl+shift+n new window
    ctrl+shift+w close window
